Data Indonesia

Data Propinsi, kabupaten, kecamatan dan kelurahan di Indonesia dalam bentuk json

Source Code

Struktur Data

  • propinsi.json
  • kabupaten/[id propinsi].json isinya data kabupaten sesuai id propinsi
  • kecamatan/[id kabupaten].json isinya data kecamatan sesuai id kabupaten
  • kelurahan/[id kecamatan].json isinya data kelurahan sesuai id kecamatan

Saat awal biasanya request dulu propinsi.json, ditambahkan ke autocomplete

Saat propinsi dipilih misal ACEH dengan id 11, lalu rekues ke folder kabupaten untuk mendapatkan daftar kabupaten. contoh kabupaten/11.json

Saat kabupaten dipilih misal KABUPATEN SIMEULUE dengan id 1101, lalu rekues ke folder kecamatan untuk mendapatkan daftar kecamatan. contoh kecamatan/1101.json

Saat kecamatan dipilih misal TEUPAH SELATAN dengan id 1101010, lalu rekues ke folder kelurahan untuk mendapatkan daftar kelurahan. contoh kelurahan/1101010.json

dan terpilihlah misal kelurahan LATIUNG dengan id 1101010001

Struktur ID 1101010001 = 11+01+010+001 = propinsi+kabupaten+kecamatan+kelurahan
